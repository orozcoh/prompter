Prompter Architecture

Frontend

  • • React 19 + Vite 8 + TypeScript 5.9
  • • PWA: vite-plugin-pwa + Workbox
  • • Offline-first with runtime caching
  • • Auto cache-clear on new deploy
  • • Wallet: wagmi 3 + viem 2

Backend Worker

  • • 7 REST endpoints
  • • x402 Payment Required protocol
  • • On-chain USDC verification (viem)
  • • OpenRouter AI integration
  • • Security headers + CORS middleware

Storage & Services

  • • GENERATED_IMAGES_R2: 1-day lifecycle
  • • OpenRouter API: dual-tier AI models
  • • Base L2: USDC on-chain verification
